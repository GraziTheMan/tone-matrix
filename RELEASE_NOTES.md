# Release notes

Copy the block under the version you're shipping into the Google Play Console
"What's new" field when you roll out a build. Play limits release notes to
500 characters per language, so each block is kept short. Newest first.

The CI version name is `0.1.<build number>` (the GitHub Actions run number);
note that beside each entry so you can match a Play upload to a commit.

---

## v0.1.23 — Track mixer, Follow, help

```
• New Track panel: set the volume of each track and the drums, and dial the
  delay effect up or down — great for bringing a bassline forward.
• The Follow button is back and always visible.
• Added an in-app Help & tutorial panel.
• Fixes and polish.
```

## v0.1.22 — Fuller drum kit

```
• Three new drum sounds: crash, tambourine, and clap. The kit is now seven
  lanes.
• Imported RollingTones songs bring their full percussion across.
```

## v0.1.20–0.1.22 — RollingTones import

```
• Import your old RollingTones songs: open a .tmx file from the Projects
  panel and it loads with melody, drums, and tempo intact.
• Nine melodic instruments and more scales, including the classic modes.
```

## v0.1.x — First release

```
Sonic Squares is a grid music maker: paint notes on the grid, layer three
melody tracks and a drum kit, chain patterns into songs, and export your
music as MIDI or WAV. Everything runs offline on your device.
```
